### 📚 递归

思维要点：

1. 不要人肉进行递归（最大误区）
2. 找到最近最简方法，将其拆解成可重复解决的问题（重复子问题）
3. 数学归纳法思维

递归代码模板：
（验证方法）

1. 保证 n = 1, n =2 时，结果正确
2. 可以证明当 n 成立时，推导出来的 n + 1 也成立

```js
const recursion = (level, params) {
  // recursion terminator 递归终结条件
  if (level > MAX_LEVEL) {
    process_result
    return
  }

  // process current level 处理当前层逻辑
  process(level, params)

  // drill down 下探到下一层
  recursion(level + 1, parms)

  // clean current level status if need 清理当前层
}
```

### 🤔 DFS 深度优先搜索 代码模板

搜索顺序：以树为例，从根节点开始，先搜索左子树，一直走左子树走到底，再搜索右子树。

- 推荐的递归写法

```js
// 递归写法
const visited = new Set();
const dfs = (node) => {
  if (visited.has(node)) return;
  visited.add(node);
  dfs(node.left);
  dfs(node.right);
};
```

- 非递归写法

```js
const dfs = (node) => {
  if (!node) return;
  const visited = [];
  const stack = node;
  while (stack.length > 0) {
    const node = stack.pop();
    visited.push(node);
    process(node);

    const nodes = generate_related_nodes(node);
    stack.push(nodes);
  }
};
```

### 🏫 经典题目

#### 22.括号生成

给定 n 对括号，请编写一个函数，用于生成所有可能的并且有效的括号组合）如((()))、()()()、(())()。每个格子都可以放左括号或者右括号

```js
const generateParenthesis = (n) => {
  const res = [];
  const walk = (l, r, str) => {
    if (str.length === 2 * n) {
      return res.push(str);
    }

    if (l < n) {
      walk(l + 1, r, str + "(");
    }
    if (r < l) {
      walk(l, r + 1, str + ")");
    }
  };
  walk(0, 0, "");
  return res;
};
```
