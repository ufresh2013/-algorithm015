# 刷 LeetCode 经常会用到的 Map, Set，String, Array 的原生方法。

### 1. Map

Map 对象是键值对的集合，其键和值可以是任意类型。

```js
const map = new Map();
map.set("key", "value");
map.get("key");
map.has("key");
map.delete("key");
for (let [key, value] of map) {
  console.log(key, value);
}
map.forEach((value, key) => {
  console.log(key, value);
});
```

### 2. Set

Set 对象允许你存储任何类型的唯一值，无论是原始值还是对象引用。

```js
const set = new Set();
set.add("value");
set.has("value");
set.delete("value");
for (let value of set) {
  console.log(value);
}

// set转换为数组
Array.from(set);
[...set];

// 数组转换为set
const set = new Set([1, 2, 3]);

// 交集
const intersection = new Set([...set1].filter((x) => set2.has(x)));
```

### 4. Array

```js
const arr = [1, 2, 3];
arr.push(4);
arr.pop();
arr.shift();
arr.unshift(0);
arr.slice(1, 3);
arr.splice(1, 1);
arr.indexOf(2);
arr.lastIndexOf(2);
arr.reverse();
arr.sort((a, b) => a - b);
arr.map((x) => x * 2);
arr.filter((x) => x > 2);
arr.reduce((acc, x) => acc + x, 0);
arr.every((x) => x > 2);
arr.some((x) => x > 2);
arr.find((x) => x > 2);
arr.findIndex((x) => x > 2);
arr.forEach((x) => console.log(x));
arr.join(",");
arr.concat([4, 5]);
arr.flat();
arr.fill(0);
arr.includes(2);

// splice(索引，删除几个元素，并插入什么元素)
// 添加或删除元素
arr.splice(1, 1); // 删除第二个元素
arr.splice(1, 0, 6, 7); // 在第二个位置插入6,7

// slice
// 截取数组，并返回新数组，不会改变原数组
// string也有这个方法
arr.slice(1, 3); // [2,3] 从索引1开始，到索引3结束（不包括索引3）
```
