# 190.颠倒二进制位
颠倒给定的 32 位无符号整数的二进制位。

输入: 00000010100101000001111010011100
输出: 00111001011110000010100101000000

```js
// 1. 需要先获得二进制数的末位，通过 1 & n 实现。已经知道 1 & 0 = 0, 1 & 1 = 1
// 2. 获取末位只后，需要获得倒数第二位数值，这里通过移位实现 n >> 1，倒数第二位移至末位
// 3. 我们只需异动32次，就能获得n的所有二进制位值
const reverseBits = (n) => {
  let result = 0
  for (let i = 0; i < 32; i++) {
    result = (result << 1) + (n & 1)
    n = n >> 1
  }
  return result >>> 0
}
```
