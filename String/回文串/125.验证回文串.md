# 125.验证回文串
```js
var isPalindrome = function(s) {
  let r = s.replace(/[^0-9a-zA-Z]/g, '').toLowerCase().split('')
  return r.join('') === r.reverse().join('')
};
```

**双指针**
```js
var validPalindrome = function (str) {
  let l = 0, r = str.length - 1; // 头尾指针
  while (l < r) { 
    if (str[l] != str[r]) {
      return false
    }
    l++;
    r--;
  }
  return true;
}
```
