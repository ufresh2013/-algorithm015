### 704. 二分查找
二分查找的前提：
① 目标函数单调性（单调递增或者递减）
② 存在上下界
③ 能够通过索引访问
```js
var binarySearch = function(nums, target) {
  let left = 0;
  let right = nums.length - 1;
  while(left <= right) {
    let mid = right + left >> 1
    if (nums[mid] === target) {
      return mid
    } else if (nums[mid] < target) {
      left = mid + 1
    } else {
      right = mid - 1
    }
  }
  return -1
};
```
